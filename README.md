# desktop-overlay
a simple overlay to show images or animations over your desktop and apps.
---

# Simply Press "Add Image To Overlay" To Add A New Image
## Currently Supported File Formats:
- png
- jpg/jpeg
- gif
- svg
---

# Once A New Image Has Been Added, Right Click It To Open The Custimization Menu
